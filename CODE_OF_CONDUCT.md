# Code of Conduct

This project follows the [Contributor Covenant](https://www.contributor-covenant.org/version/2/1/code_of_conduct/),
version 2.1. In short: be respectful, be welcoming, and assume good faith. Harassment,
discriminatory language or jokes, and personal attacks aren't tolerated in issues, pull requests,
discussions, or any other project space.

## Reporting

If you experience or witness unacceptable behavior in this project, report it by opening a
[GitHub issue](https://github.com/JGalego/TeamAPI/issues) or, for sensitive reports, via GitHub's
[private vulnerability reporting](https://github.com/JGalego/TeamAPI/security/advisories/new)
form. Reports will be reviewed and handled with discretion.
